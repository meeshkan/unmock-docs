Installation

Note: unmock documentation uses yarn, but npm will also work. You can compare yarn and npm here.

CLI

Global installation

Yarn

yarn global add unmock

npm

npm install -g unmock

Homebrew

brew tap unmock/unmock
brew install unmock

Install in project

Yarn

yarn add unmock -D

Execute with

yarn unmock

npm

npm install unmock --save-dev

To execute, install npx

npm install -g npx
npx unmock

or

./node_modules/bin/unmock

Node.js

Jest

Install jest-unmock:

yarn add -D jest-unmock

Add jest-unmock as preset injest.config.json:

// jest.config.json
{
  "preset": ["jest-unmock"]
}

or in package.json:

// package.json
"jest": {
  "preset": ["jest-unmock"]
}

If you're already using another preset like ts-jest, add jest-unmock in setupFilesAfterEnv:

// package.json
"jest": {
  "setupFilesAfterEnv": ["jest-unmock/jest-setup.js"],
}

Now whenever Jest runs, unmock will capture all outgoing HTTP requests and serve local mocks from __unmocks__ folder.
